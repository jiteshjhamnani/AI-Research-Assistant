# AI-Research-Assistant

An AI-powered research assistant built using LangGraph, LangChain, Groq LLM, Tavily Search, Wikipedia, and Arxiv.

This application performs multi-step research by planning the query, gathering information from trusted sources, and generating a concise structured response.
  
Tech Stack :-  Python,Streamlit,LangGraph,LangChain, Groq API (Llama 3.1) , Tavily Search API , Wikipedia API , Arxiv API
